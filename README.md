# Yes Bank Stock Analysis

## Project Overview
This project focuses on analyzing historical Yes Bank stock price data to understand trends, volatility, and price behavior.

## Objectives
- Perform exploratory data analysis on stock price data
- Identify trends and volatile periods
- Study relationships between Open, High, Low, and Close prices
- Generate insights to support investment and risk analysis

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Yes Bank stock shows high volatility across certain time periods
- Strong correlation observed between price variables
- Outliers indicate high-risk market phases
- Data-driven insights can support financial decision-making
